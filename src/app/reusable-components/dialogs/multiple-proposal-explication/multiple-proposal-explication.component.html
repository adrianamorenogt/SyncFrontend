Explicación propuesta multiple
Para hacer una propuesta multiple debes subir un archivo de formato CSV donde cada elemento se separe usando comas (","), dicho archivo debe seguir las siguientes especificaciones.
Campos requeridos:
  1. Tipo y numero de Documento: Numero de documento del cliente al cual se le esta haciendo la propuesta de negociación
  2. Agencia o entidad financiera: Entidad o agencia de cobranza con la cual el cliente tiene la deuda.
  3. Monto: Monto por el cual se planea negociar la deuda del cliente.
  4. Descripción: Detalle de la propuesta, en este campo se puede especificar la forma en que se va a negociar la deuda del cliente.
  5. Referencia: La referencia asociada a la deuda, en caso de conocerse.

  • Propuesta dentro del CSV:
